MyRuns
MyRuns is an Android app I developed using Java and XML over six weeks as a part of my Smartphone Programming class that uses Google Maps to track your movements while exercising, and Firebase to store your data. The development of the app was broken into four different assignments. First, I created authentification and saved profiles with photo storage. For the second assignment, I finished the UI and added the ability to manually add exercises. For the third assignment, I implemented the Firebase database. Finally, for the last assignment I added the Google Maps based tracking feature. This project is available on my Github.
